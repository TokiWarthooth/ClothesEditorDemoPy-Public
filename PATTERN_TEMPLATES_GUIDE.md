# Pattern Templates Guide

## Обзор

Система шаблонов выкроек позволяет быстро добавлять готовые элементы одежды на холст с настраиваемыми параметрами.

## Доступные шаблоны

### 1. Рукава (Sleeves)
- **Sleeve** - базовый рукав с окатом
  - Width (ширина): 100-300 px
  - Height (длина): 200-600 px
  - Curve Depth (глубина оката): 10-60 px

### 2. Воротники (Collars)
- **Collar** - классический воротник
  - Width (ширина): 100-400 px
  - Height (высота): 40-150 px
  - Neck Curve (вырез горловины): 20-80 px

### 3. Карманы (Pockets)
- **Pocket** - прямоугольный карман с закругленными углами
  - Width (ширина): 60-200 px
  - Height (высота): 80-250 px
  - Corner Radius (радиус углов): 5-40 px

### 4. Юбки (Skirts)
- **Skirt Panel** - панель юбки (передняя/задняя)
  - Waist Width (ширина талии): 100-300 px
  - Hip Width (ширина бедер): 150-400 px
  - Length (длина): 200-800 px

### 5. Брюки (Trousers)
- **Trouser Leg** - штанина
  - Waist Width (ширина талии): 120-350 px
  - Hip Width (ширина бедер): 150-400 px
  - Leg Width (ширина низа): 100-300 px
  - Length (длина): 500-1200 px

### 6. Лиф (Bodice)
- **Bodice Front** - передняя часть лифа
  - Width (ширина): 150-350 px
  - Length (длина): 250-600 px
  - Shoulder Width (ширина плеча): 80-200 px
  - Neck Depth (глубина выреза): 40-150 px

## Как использовать

1. **Выбор инструмента Pattern**
   - Нажмите на кнопку "Pattern" в панели инструментов
   - Откроется панель "Pattern Templates" слева

2. **Выбор шаблона**
   - Выберите категорию из выпадающего списка (Sleeves, Collars, и т.д.)
   - Выберите конкретный шаблон

3. **Настройка параметров**
   - Используйте слайдеры для настройки размеров
   - Значения отображаются над каждым слайдером

4. **Размещение на холсте**
   - Нажмите кнопку "Add to Canvas"
   - Кликните на холсте в нужном месте для размещения шаблона

5. **Перемещение шаблонов**
   - Кликните на размещенный шаблон и перетащите его в нужное место

## Советы

- Используйте сетку (Grid) для точного позиционирования
- Шаблоны можно комбинировать для создания полных выкроек
- Каждый шаблон можно добавить несколько раз с разными параметрами
- Используйте инструмент Line или Bezier для доработки шаблонов
