# Roadmap — Clothing Designer Demo

Цель: за 2 месяца (к ~2026-08-07) собрать рабочий редактор выкроек
с экспортом в форматы, которые поддерживают Clo и Valentina.

---

## Форматы экспорта (ориентир)

| Формат | Clo | Valentina | Приоритет |
|--------|-----|-----------|-----------|
| SVG    | да  | да        | 1 — самый простой, встроен в Qt |
| PDF    | да  | да        | 2 — QPrinter, разбивка на A4 |
| DXF    | да  | да        | 3 — нужна библиотека `ezdxf` |
| PNG    | да  | да        | бонус — тривиально через QImage |

> DXF — стандарт для раскройных машин. Без него демо неполное.

## Фазы разработки

### Фаза 1 — Базовый рабочий редактор (недели 1–3)

Делаем из "багнутого paint" нормальный инструмент.

- [x] **Система измерений** ✓
  - Шкала пикселей → сантиметры (1 см = 37.795 px при 96 dpi)
  - Индикатор координат в статус-баре (X: 12.3 см, Y: 4.5 см)
  - Переключатель единиц: cm / mm / px (комбобокс в статус-баре)
  - Линейки по краям холста (горизонтальная + вертикальная) с индикатором курсора

- [x] **Зеркальное отражение деталей** ✓
  - По горизонтали: Edit → Flip Horizontal (Shift+H)
  - По вертикали: Edit → Flip Vertical (Shift+V)
  - Работает на любом выделенном объекте, повторный flip возвращает оригинал

- [ ] **Улучшение Select-инструмента**
  - Выбор объектов кликом (сейчас не работает нормально)
  - Рамочное выделение (drag to select)
  - Delete / Backspace для удаления

- [x] **Undo / Redo** ✓
  - QUndoStack в canvas, команды: AddItem, RemoveItems, Transform
  - Ctrl+Z / Ctrl+Y через Edit меню (текст кнопок меняется: "Undo Draw line" и т.д.)
  - Охват: линии, кривые Безье, шаблоны, удаление, flip

### Фаза 2 — Профессиональные инструменты (недели 4–6)

Фичи, отличающие редактор выкроек от обычного редактора.

- [ ] **Припуски на швы (Seam Allowance)**
  - Офсет контура выбранной детали на N мм
  - `QPainterPathStroker` — уже есть в Qt
  - Настраиваемое значение (обычно 10–15 мм)
  - Визуальное отображение пунктиром

- [ ] **Обозначения швов**
  - Стиль линии: прямой, зигзаг, оверлок (через QPen dash pattern)
  - Выбор в панели свойств

- [ ] **Аннотации**
  - Текстовые метки (`QGraphicsTextItem`)
  - Стрелки направления долевой нити
  - Нумерация деталей

- [ ] **Слои**
  - Панель слоёв: добавить, удалить, скрыть, заблокировать
  - Порядок (поднять/опустить)
  - `QGraphicsItemGroup` как основа

### Фаза 3 — Экспорт (недели 7–8)

Делаем выкройку пригодной для реального использования.

- [ ] **Экспорт SVG**
  - `QSvgGenerator` + `QPainter`
  - Сохранение всех объектов сцены
  - Единицы в мм (для совместимости с Valentina)

- [ ] **Экспорт PDF**
  - `QPrinter` с форматом A4
  - Автоматическая разбивка большой выкройки на листы
  - Метки для склеивания листов

- [ ] **Экспорт DXF**
  - Библиотека `ezdxf` (pip install ezdxf)
  - Линии и кривые Безье → LWPOLYLINE / SPLINE
  - Слои DXF = слоям проекта

- [ ] **Экспорт PNG**
  - `QImage` + `QPainter` на сцену
  - Выбор разрешения (72 / 150 / 300 dpi)

## Что оставить за рамками демо

| Фича | Причина |
|------|---------|
| Симуляция драпировки | Отдельный физический движок, уровень Clo 3D |
| Расчёт расхода ткани | Алгоритм нестинга (NP-сложная задача) |
| Выбор типа ткани с симуляцией | Профессиональный уровень, требует месяцев |
| 3D-просмотр | Отдельное приложение |

## Текущее состояние (старт)

- Инструменты: Pen, Line, Bezier, Rectangle, Ellipse, Select (частично), Pattern
- Шаблоны выкроек: Sleeve, Collar, Pocket, Skirt, Trouser, Bodice
- Сетка с привязкой, темы (свет/тёмная)
- Save/Load (.cld JSON) — базовая версия без сериализации объектов холста
- Undo/Redo: нет
- Экспорт: нет
- Измерения: нет
